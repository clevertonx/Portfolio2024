Tecnologias

Descubra as habilidades e ferramentas que domino e que me permitem criar soluções criativas e funcionais para meus clientes.

  • Desenvolvimento Java

    Java é uma linguagem de programação versátil e robusta, amplamente utilizada para desenvolvimento de software.

  • Spring Boot

    O Spring Boot simplifica o desenvolvimento Java, proporcionando configurações padrão para criar aplicativos de forma rápida e eficiente.

  • Html5

    HTML5 é a última versão do HTML, usado para criar conteúdo em páginas web, com suporte a áudio/vídeo e elementos semânticos. Essencial para o desenvolvimento web.

  • CSS

    CSS3 é a terceira versão do CSS, usado para estilizar páginas HTML. Introduz recursos avançados como sombras, transições e animações para design moderno e responsivo.

  • JavaScript

    JavaScript é uma linguagem para tornar páginas web interativas, executada nos navegadores, sendo essencial para o desenvolvimento web front-end.

  • React

    React é um framework de JavaScript para criar interfaces de usuário interativas e reativas de maneira eficiente.

  • MySql

    MySQL é um sistema de gerenciamento de banco de dados relacional de código aberto, usado para armazenar e recuperar dados eficientemente usando SQL.

  • PostgreSql

    O PostgreSQL é um banco de dados relacional de código aberto amplamente usado em projetos empresariais devido ao seu suporte completo a SQL, transações seguras e alta escalabilidade.
